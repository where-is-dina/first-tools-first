## What issues I ran into — and how I solved them

### 1. Couldn't find the Extensions panel
The instructions said to go to Extensions to install
Claude Code and Codex. My version of Cursor had a
completely redesigned UI — an Agent chat view and a
Marketplace instead of the traditional extensions
sidebar.

**How I solved it:** Discovered that pressing Cmd+Shift+X
only works from the code editor view, not the agent chat
view. Switched views and the Extensions panel appeared.

### 2. Codex and Claude were already built in
When I saw Codex 5.3 and Claude Sonnet 4.7 already
in the interface, I got a little confused — were these
the same as the extensions I was supposed to install?
I wasn't sure if they counted or if I still needed to
do something.

**How I solved it:** I didn't overthink it. I kept
navigating Cursor, looked deeper into the interface,
and it clicked. Ended up installing both through
Extensions anyway just to confirm the process —
and boom, it worked.

### 3. Git was not installed on my Mac
When I ran git clone for the first time, Terminal
prompted me to install Git via Xcode Command Line Tools.

**How I solved it:** Clicked Install on the macOS
prompt, waited for it to complete, then re-ran the
git clone command successfully.
